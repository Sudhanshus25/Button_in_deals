CRM Deal Toolbar Button App

How to Install and Use

  1. Create a new application in your Bitrix24 portal
  2. Add this URL as a placement handler
  3. Select the placement type: CRM_DEAL_DETAIL_TOOLBAR
  4. Configure the necessary permissions (CRM access)
  5. Install the application in your Bitrix24
